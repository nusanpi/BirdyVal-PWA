# Species data

The production species database (`aves_cv_enriched.json`, ~700 species of the Comunitat Valenciana) is **not included** in this repository. Its descriptions are derived from material kindly provided by the [Societat Valenciana d'Ornitologia (SVO)](https://svornitologia.org/) and cannot be redistributed.

To run the app locally, use the included sample:

```bash
cp aves_cv_enriched.sample.json aves_cv_enriched.json
```

The sample contains 5 species with original descriptions and follows the exact same schema as the full dataset.
